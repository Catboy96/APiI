Time

APIs for time.


date.sh

Returns system current date and time.

GET /stat/time/date.sh => JSON:
{"result":"Fri 12 Jul 2019 10:16:57 AM CST"}


upsince.sh

Returns system uptime since which time.

GET /stat/time/upsince.sh => JSON:
{"result":"2019-07-12 08:11:02"}


uptime.sh

Returns system uptime.

GET /stat/time/uptime.sh => JSON:
{"result":"up 2 hours, 6 minutes"}
