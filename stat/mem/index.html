Memory

APIs for RAM statistics.


app.sh

Returns the size of RAM used by applications. (result in KBytes)

GET /stat/mem/app.sh => JSON:
{"result":133796}


avail.sh

Returns available RAM size. (result in KBytes)

GET /stat/mem/avail.sh => JSON:
{"result":837264}


buff.sh

Returns the size of RAM used by buffers. (result in KBytes)

GET /stat/mem/buff.sh => JSON:
{"result":52612}


cache.sh

Returns cached RAM size. (result in KBytes)

GET /stat/mem/cache.sh => JSON:
{"result":329392}


free.sh

Returns free RAM size. (result in KBytes)

GET /stat/mem/free.sh => JSON:
{"result":484512}


total.sh

Returns total RAM size. (result in KBytes)

GET /stat/mem/total.sh => JSON:
{"result":999040}
