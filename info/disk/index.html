Disk

APIs for disk information.


part.sh ⚡

Returns the partition details of all stroage devices.

GET /info/disk/part.sh => JSON:
{"result":[{"device":"/dev/mmcblk0p1","start":"8192","end":"532480","sectors":"524289","size":"256M","id":"c","type":"W95 FAT32 (LBA)"},{"device":"/dev/mmcblk0p2","start":"540672","end":"31137791","sectors":"30597120","size":"14.6G","id":"83","type":"Linux"}]}


usage.sh

Returns disk usage of all stroage devices.

GET /info/disk/usage.sh => JSON:
{"result":[{"fs":"/dev/root","size":"15G","used":"1.3G","avail":"13G","usep":"10%","mounted":"/"},{"fs":"/dev/mmcblk0p1","size":"253M","used":"40M","avail":"213M","usep":"16%","mounted":"/boot"}]}


⚡: API requires user "www-data" in sudoers file with no-password enabled.
