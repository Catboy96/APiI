CPU

APIs for CPU information.


arch.sh

Returns CPU architecture.

GET /info/cpu/arch.sh => JSON:
{"result":"armv7l"}


bogomips.sh

Returns CPU BogoMIPS.

GET /info/cpu/bogomips.sh => JSON:
{"result":"38.40"}


cores.sh

Returns CPU cores count.

GET /info/cpu/cores.sh => JSON:
{"result":"4"}


hardware.sh

Returns CPU hardware information.

GET /info/cpu/hardware.sh => JSON:
{"result":"BCM2835"}


model.sh

Returns CPU model information.

GET /info/cpu/model.sh => JSON:
{"result":"ARMv7 Processor rev 4 (v7l)"}


revision.sh

Returns CPU's revision.

GET /info/cpu/revision.sh => JSON:
{"result":"a22082"}


serial.sh

Returns CPU's serial number.

GET /info/cpu/serial.sh => JSON:
{"result":"00000000abcd1234"}


minfreq.sh ⚡

Returns CPU's minimum frequency.

GET /info/cpu/minfreq.sh => JSON:
{"result":600000}


maxfreq.sh ⚡

Returns CPU's maximum frequency.

GET /info/cpu/maxfreq.sh => JSON:
{"result":1200000}


curfreq.sh ⚡

Returns CPU's current frequency.

GET /info/cpu/curfreq.sh => JSON:
{"result":800000}


⚡: API requires user "www-data" in sudoers file with no-password enabled.
