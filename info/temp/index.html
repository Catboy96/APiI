Temperature

APIs for temperature information.


cpu.sh

Returns current CPU temperature.

GET /info/temp/cpu.sh => JSON:
{"result":"48'C"}


gpu.sh ⚡

Returns current GPU temperature.

GET /info/temp/gpu.sh => JSON:
{"result":"48.9'C"}


⚡: API requires user "www-data" in sudoers file with no-password enabled.
